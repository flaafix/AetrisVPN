# Автоматический белый список (Whitelist) для VPN

Репозиторий содержит актуальные списки доменов и IP-адресов, которые направляются в обход VPN-соединения (прямое подключение). Все данные обновляются автоматически **каждый час**.

---

## 🚀 Особенности проекта

* **Частота обновления:** Каждый час (24/7) без участия человека.
* **Автоматизация:** Сборка и проверка данных работают на базе GitHub Actions.
* **Оптимизация:** Списки очищены от дубликатов и готовы к быстрой загрузке.

> [!NOTE]
> Списки обновляются на стороне GitHub. Если вы один раз добавили ссылку в свое приложение, вам больше не нужно ничего настраивать вручную — клиент сам скачает свежую версию.

---

## 🛠 Инструкция по подключению

Для настройки вашего VPN-клиента используйте прямые ссылки (`Raw`).

### Как получить ссылку:
1. Откройте нужный файл списка в этом репозитории.
2. В верхнем правом углу текстового блока нажмите кнопку **Raw**.
3. Скопируйте получившийся URL-адрес из строки браузера.

### Поддерживаемые клиенты:
* **Sing-box / Xray / v2ray:** Скопированная ссылка вставляется в блок удаленных правил (`remote rules`).
